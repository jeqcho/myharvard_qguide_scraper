What would you like to tell future students about this class? (Your response to this question may be published anonymously.)

Table for [QTitle].
Comments
This class is hard. There's a lot of readings and you're definitely going to read something outside your wheelhouse. But the class rewards the effort; the environment is one in which you can still try to contribute even as you're feeling your way through the material; and, most importantly, the material is super interesting. So be prepared to put in the work and to be rewarded for putting in the work.
If you are interested in linguistics from a experimental/developmental perspective, this is a great class to take. However, just because it meets once a week, it is not a light class — it is predominantly a graduate seminar.=. The readings are long and difficult but I found all of them interesting and found myself thinking about many of the topics all the time outside of class; in fact, so much so that I turned a friend into a prospective linguistic concentrator as I was talking about the topics so much. An introductory psych or linguistics class would help before taking this course as there are some very foundational things that are expected to be understood in discussion without being glossed. Otherwise, be willing to do some extra reading outside of the class content to familiarize oneself with some of the gaps.
This class is a great opportunity to get exposure to two different fields and the intersection between them.
