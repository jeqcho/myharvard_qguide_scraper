Since you have to take an Expos class anyways, then this is a good option. Overall, pretty good, would recommend.

Take this class if you enjoy literature, but be prepared for some heavier reading assignments towards the middle of the semester.
Dr. Sweeney is great, and this topic is very engaging! However, it is definitely more work than some other sections of expos; though the third unit is a movie unit, you're expected to watch and analyze all three in a few days, and Uncle Tom's Cabin is the entirety of the second unit. I did really learn a lot so I would recommend it with enthusiasm, but I'd warn future students to commit to the class if and only if they understand it'll be a time commitment. It really is worth it though – I feel like I'm a much better writer after!
This is a fantastic Expos course that gives you back exactly what you put into it. If you make the most of this class, you will not only learn how to write rigorously but also interesting readings and fascinating discussions. Dr. Sweeney truly cares for and tries to do the best for all his students, and he is so understanding about any issues that you may be having and is willing to work with you in order to find solutions for these problems. While the course is a little rigorous in its pace, Dr. Sweeney guides the class discussion in a way that makes it far easier to 'keep up' with and takes the time to check in with the class to see how everyone is feeling. He is extremely fair in his grading and gives copious reasoning for the grade that he gives. This course truly taught me how to write at the college level, and I think that it was one of the most enjoyable experiences so far at Harvard. This is truly a fantastic Expos course!
The content itself is interesting, but this class requires writing in a kind of abstract way, which is something I was extremely unfamiliar with, so it was quite difficult for me. If you really enjoy writing in any style though, this class is AMAZING! Dr. Brian Sweeney is also super sweet and understanding.
Sweeney is a really great discussion facilitator, super nice, and won't cold call you for the most part if he see's you aren't in an amazing mood. However, this expos has WAY more mandatory readings and assigments than most others. There were 2–4, 20–40 page readings and 100+ word responses to each, for each week. That was Unit 1. Unit 2 was a 500 page novel read over 3–4 weeks, with the same responses each week. Unit 3 was just as bad even though Prof. Sweeney didn't think so. Overall, it was a very constructive class with great discussions, but it was way too much work for an expos class where others average 0–1 hours of work a week besides essays.
