What would you like to tell future students about this class? (Your response to this question may be published anonymously.)

Table for [QTitle].
Comments
Overall, I think I would recommend this first year seminar. I found the content of the course very interesting and the discussions to be wide–reaching. It was also a very laid back course but remained very interesting. My only real critique is that I think the course would have benefitted from more student driven conversations.. I think there is a lot of value in professor led discussion but I also think it would've been cool to be able to talk more directly student to student.
Dr. Haig is a very kind and understanding professor that is truly passionate about the topic. He genuinely wants each student to be involved in discussion and is invested in each person in the room (including him) learning from and getting something out of the conversation. He also has decades of experience at Harvard which come with invaluable wisdom.
This class had a light workload and a very kind professor. It wasn’t the most engaging class I’ve ever taken, but there were some very interesting themes and I don’t regret taking it. Great seminar setting, no work other than a weekly reading, and meets once a week. If you’re interested in genetics and evolution, this class is worth checking out!
I would really recommend this course as Dr. Haig is an incredible teacher and person.
