This is a great class. The coursework is pretty low pressure and the material is interesting, and the professor is really nice and understanding. I think, since this was the first version of this course taught by this professor, it was a bit disorganized and the expectations for assignments were a bit unclear, but I believe that will be resolved in future iterations.
***This is a great class! Robert is the best. He knows his stuff and he is very approachable if you ever have questions or concerns. Material is interesting. Reading assignments are doable and there isn't too much busywork. The sociolinguistic variation project is super cool and you will learn a lot. 2 classes a week, no section, makes for a perfect elective if you have any interest in language & social interaction.
I really enjoyed this class. Take especially if you have an interest in ling or socio. Prof Xu is pretty chill, and he is really passionate about the material. Which shows in his teachings. I would say this was prob one of the most interesting classes I have taken so far.
This class is so interesting, and the topic is very fun! You get to learn about different aspects of sociolinguistics, including language in the context of politics, personal relationships, education, and more. The slides and class time are engaging and interesting, and the work outside of class is the same.
If you have an interest in how language interacts with and from society (gender, race, dialects, geography, history, technology, etc etc), this will be a fun course for you! Pretty accessible to beginners, even those with no background in linguistics. This course is very very new in the department so it's still working out some course logistics, but I'm confident the subsequent versions of this course will be strong. Robert is a highly approachable professor, and genuinely enjoys teaching this content. Assignments and projects definitely required time and effort, but were very manageable. A great addition to the linguistics department, and a great choice for anyone interested in these concepts. That being said, the class is pretty introductory, so if you have an extensive background in these concepts, some of the content might feel rudimentary or repetitive. However, the course does provide additional depth to these topics, so maybe talk to Robert first to figure out if this course is right for you. But if you're someone who already enjoys sociolinguistics and dabbles in learning about this for fun, you'll definitely enjoy this course even if it covers something you already know.
TAKE THIS CLASS!!!! Whether you're interested in linguistics or not, this is a super fun and engaging class. It is 9am so sometimes that component is tough, but trust, it is worth going to class because the material and discussions were always very insightful. Robert is a great teacher! Workload is fair, but assignment feedback was not the greatest (and the only real complaint I have). I only had 2 assignments graded the entire semester so it was really hard to gage how I was actually doing in the class and whether I needed to put more effort into the assignments. But aside from that, super fun class and highly recommend.
This is a good introductory class for ling if you are interested in learning if you'll like linguistics.
