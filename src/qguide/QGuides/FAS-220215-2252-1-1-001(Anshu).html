What would you like to tell future students about this class? (Your response to this question may be published anonymously.)

Comments
Come prepared with questions to class if you have been confused about a topic, even if it’s from previous lectures.
The class is a decent intro to quantum info with great breadth. The main issue, however, is that lectures are often difficult to follow, lecture notes are incomplete, and you end up internalizing little of the actual information. The psets also test basic linear algebra skills rather than content covered in class, which made the course easy to get through without internalizing the content.

If you want to learn more about quantum computing, this class is a fair way to do it at a great breadth. Nielsen and Chuang is also a great resource, but doesn't quite reach the breadth of what this course covers. However NC probably explains basic concepts like phase estimation much better than this class did.
I thought this class is great in terms of content; however, I would caution students on taking a course in quantum info before this one as it is very fast–paced and foundational concepts are not covered in detail. This class is the theoretical CS version of quantum info and a strong background in linear algebra and modular arithmetic would be helpful to have. Overall, this class did not take too much time out of my schedule and I thought the concepts and topics were pretty interesting!
Do not take this class if you have only a physics background.
