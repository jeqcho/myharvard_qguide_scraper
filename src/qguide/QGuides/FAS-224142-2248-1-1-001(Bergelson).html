Great class! I am not a psych major at all, but I learned so much from the class and everyone is so passionate about the topics at hand. It is definitely a lot of work which is why I am glad I took it during a more chill semester, but it is definitely a class that is worth taking because Dr. Bergelson is so passionate about this research. It was easily the most interesting class I've taken at Harvard.
I loved this class! My passion for the subject matter definitely played a role in my enjoyment of the course, so I think it's important you have some interest in linguistics, child language development, or language sciences in general if you decide to enroll.
this class was a ton of work. lots of reading and it had to be thorough to engage in class discussion/be a section leader. Dr. B is very intelligent but I think the course either should have been for grad students or modified to fit a "non–expert" view that we have as undergrads. the content was super interesting and i loved the people in my class (bc we bonded over how much work it was) but i think some modification or some leisure in grading needs to be implemented. This is a good class to take pass/fail if you're interested in the subject matter but don't want to worry too much about your grade as it is a ton of work. We had like 4 big assignments due within 1 week of each other right after thanksgiving which was horrible
Don't take this class unless this is something you are very advanced in or actually interested in linguistics or developmental psychology. Too many readings a week, so don't take it unless you have a good amount of free time. Not like many other fun and interesting psychology classes. I wanted to take it because I wanted to learn more about myself as a bilingual student, and learn more on how to help raise children in a bilingual environment, but not what I was expecting. Several people dropped out of the class because it was a lot of work and not very flexible to those who were not professionals in this subject.
This course would be helpful if you are interested in learning about language and development. The workload is meaningful and you will receive more preference in the class if you are familiar with the material. Taking the class as a member of the lab is recommended.
By far the best class I took this semester.

Dr. B is a fantastic professor and a great human being, she's incredibly knowledgeable about the field and effectively steers discussions into interesting places when necessary.

The material itself is also top–notch, you can tell a lot of thought went into selecting the papers for the class and the workload is super reasonable, definitely not an easy class if you want to do well and take away a lot from it, but also not an incredibly difficult class by any means.

The discussions are great, it benefits a lot from the very small class size, and all assignments are very feasible & communicated well in advance. The only non–fun part of the class is the biweekly mandatory reading questions, but, as Dr. B says a previous student wrote, "if we didn't have to do those I wouldn't have done the readings", which is absolutely true, so completely fair in hindsight (and when it was clear the questions were too time–consuming, the prof did decrease them, so also very receptive to feedback).

10/10 class, would definitely take again, if I could take seminars with Dr. B for the rest of my time here, I would. My guess is that the lottery for this will become quite competitive over the next few years, so get in early!
