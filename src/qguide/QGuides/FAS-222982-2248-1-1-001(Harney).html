This class is pretty good. It's a surprising amount of work so don't let it catch you off guard. You can get by without doing the readings but they're there if you want them. Eaodain is super friendly and supportive, so it makes you want to do well to just match her energy. It's really fun to stick with an unknown genome and characterize it over the course, but make sure to not leave everything until the last minute because sometimes the Jupyter notebook coding is a pain. But writing weekly practicals really helps you become a better scientific writer by seeing what's important to include or what not to write about.
I took this course as an elective, and I loved it! Professor walks you through each class day with a lot of details. You can tell that she really cares and is so well–prepared based on how extensively detailed each slideshows are. And since the class is so small, it's good to pay attention to her presentations :') There is a practical due each week, and they do not take long (the professor tells you exactly what to write for them), but do them with a lot of thought. You can get points taken off for missing small details, so just make sure you cover everything. The final project is an accumulation of all of the practicals, so it is good that you have the practicals done before then so that the final paper will not take as long to write.
GREAT CLASS! * Highly recommend. Some coding experience is definitely useful if you don't want to struggle a bit the first couple weeks but you can easily get by without it. Overall, I learned a lot and got a lot of out this rather low–stress and rather low–time commitment course.
You should take this class if you're interested in modern genomics research, even if you have a strong coding ability.
Learned so much about analyzing DNA through this course. Each week involves a practical report where you investigate one aspect of your selected "mystery genome," such as its chromosomal sex, potential phenotypes, and ancestry. These reports take a bit of time to complete, but I genuinely learned so much each week and had a great time. Would recommend starting the practical reports as early as possible and avoid doing everything the night before. Also wanted to mention that Éadaoin provides line–by–line feedback for each report, which allowed me to learn so much about scientific writing. Éadaoin is also super kind and responsive over email.
This class is by no means easy – I enrolled with the expectation that it would be relatively straightforward. However, the weekly practicals are graded quite harshly and you truly need to spend a lot of time on them (I spent 9–10 hours a week on some of them, although this might've been overkill depending on how much experience you have with scientific writing) to do very well. I did feel I got a lot out of them, though. The actual analysis techniques can also be tricky if you don't have prior coding experience. Overall, I'd recommend the class – but definitely spend enough time on the practicals/asking Eadaoin questions so you do well on them.
I really enjoyed this class! It is a fair amount of work, but I learned so much about ancient DNA and genetic analysis and I think these tools will really help me in the future. Eadaoin grades assignments in a way that feels a bit tough but very fair, and she gives really great feedback to help you understand how to improve.
I came into this class with little computational experience and no knowledge in analyzing human genomes, but Eadaoin was amazing!! She crafted each lesson with just the right amount of detail, and I loved the way the class was structured such that it explained theory on Mondays while giving us a chance to practice the material on Wednesdays. She was always available during or after class to go over questions that I had. I felt overall, Eadaoin worked incredibly hard to make the class accessible for everyone (e.g. installing a package that otherwise was important for conducting analyses). For instance, upon learning that we were interested in pathogens, she invited a guest speaker to talk about their work on ancient malaria genomes. This course provided such a great foundation for computational analysis for genomes, and I really enjoyed getting to know my mystery individual over the course of the semester!
