What would you like to tell future students about this class? (Your response to this question may be published anonymously.)

Table for [QTitle].
Comments
While you may not use results from this class in your work directly, I think there is still strong value in taking this course. A deeper understanding of probability theory, and just general familiarity with certain notation (e.g., Lebesgue integrals), has greatly helped me to understand a wider range of statistical texts. This aspect of the course can make it very beneficial to researchers/scholars in the statistical field.
This course went extremely fast as we covered a lot of topics in considerable depth with wide generality (few mathematical assumptions). I came in with an "eating my vegetables" mentality, and indeed I don't know if I'll ever actually use this material. Still, if you like theoretical probability/math/measure theory, this is a good course for you.
You need to be interested in measure theoretic probability in order for this to be fun at all.

If you are, this can be quite a fun class.
If you're not interested at all, I imagine this class would feel like pulling teeth.

Rajarshi is a great instructor and I would recommend the class for anyone who wants to start understanding measure theoretic probability better, but if proving things like the Dominated Convergence Theorem, Monotone Class Theorem, Kolmogorov's Extension Theorem, etc. doesn't already sound fun or intriguing to you, then the class will probably not be fun!
