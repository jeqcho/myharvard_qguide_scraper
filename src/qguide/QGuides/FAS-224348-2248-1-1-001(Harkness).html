Great class! I expect this class will be a lot busier next year with all the positive reviews so enjoy!
This class is very good for learning anthropological analysis methods. It was a great intro, and didnt spend time on the history of the field so much as the basics of how to actually do anthropological studies. I recommend making sure you understand the first assignment, since it is really the basis for all later assignments, and it was fairly difficult for me, but the professor was very helpful and was willing to walk me through the assignment when I was very frustrated, so if you ask for help the class staff will be willing to help you out. The class was good and I would recommend it.
TAKE THIS CLASS WITH PROFESSOR HARKNESS. Even if you have no prior knowledge about anthropology, introducing you to anthropology and building your anthropology skillset is exactly the purpose of this class. This class is very low–stress, but still forces you to think deeply about social structures and institutions, the ideologies behind them, how speech and action configure into their everyday operations, as well as their underlying categories.
This course dives into case studies to translate anthropological methodology into real phenomena. The assignments are great because they let students apply topics learned in class to a subject of their choosing.
Please take this class. The assignments are broad, short and really allow you to explore your own interests. This is an excellent intro class with few hours time commitment a week. The readings are really interesting and there are some great lectures. Professor Harkness is a really engaging lecturer and its clear that he cares about his students – he's very easy to talk to. Lectures tend to be very linguistic anthropology focused so take this class particularly if you find this aspect of anthropology interesting. There are no sections (which helped a lot with class scheduling) bit there is a lot of small group discussion in class. This class really changed my approach to thinking about how I tackled writing papers for other classes. GEM!
This course has short, straightforward assignments and a good reading list. Focuses slightly on linguistic anthropology because this is the specialty of Harkness. Very accessible.
Take this class if you want to learn how to think in a completely new way! Gave this class a chance as an elective and it did not disappoint! The readings are manageable, class discussions fruitful, and the lectures are simply exceptional!
*** Good overall course with a very manageable workload. Throughout the semester we had 5 assignments of about 800 words each. In class discussions were relatively interesting, although it did feel like they were all over the place sometimes. Definitely made me think deeper about analyzing social situations in ways that I hadn't before. This was the first time it was taught, but would definitely recommend.
Take this class! It is super fun and I really loved it! It will make you think about things critically and see its salient categories and the readings were also really enjoyable! Prof. Harkness is a wonderful lecturer and knows how to keep the class engaged!
