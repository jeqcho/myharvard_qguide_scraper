If you are willing to spend effort to learn the material, this is an alright class. It does teach both R dashboards and packages, both of which are generally useful skills.

However, the class structure this year was extremely disorganized: grades were returned extremely late, lectures bounced between topics erratically, and there was a distinct lack of communication between James, the CAs, and the students. This was the first year James was teaching the class, so it might get better in future years, but lectures, assignments, and feedback were all lackluster.

I personally entered with no R background, and it was manageable, but knowing R before coming in would make the class significantly easier.

Overall, I feel like all of the material could be more efficiently learned independently, but this class is alright if needed.

This is a good course, but it is sometimes frustratingly disorganized. i LOVEDDDD this class!! maybe not everyone feels the same but i was fine with lectures (the pace / start time) and the slides being a bit disorganized. i genuinely learned sooo much and became much more proficient in R because of this class. james is awesome and included a lot of applicable skills in what we learned. he was also very receptive to what the class wanted to focus on or was struggling with. also very understanding with deadlines and course expectations. but really this class ended up being my favorite this semester. it teaches you a lot but also encourages research on your own through PSETs and the projects. it does a good job of giving you an "under the hood" type of understanding about R packages and teaches you a fair amount of coding in base R without package dependency. the projects were really enjoyable to complete and there was a good amount of office hours for support from the TFs (mally is great!!) The organization of this course year was not great, but the content was. If you have a basic background in R, you will learn how to make more complex graphs, make a dashboard (kind of like a website) to display information about data, and learn how to make your own package. I also got more comfortable using github and after taking this course I can confidently put "coding in R" and using these dashboard skills on a CV.
I would highly recommend this course if you are a statistics concentrator. This is the intro to R class you never took, but wish you had. You learn so much about specific features of R (and if you're looking for resume building skills, this course definitely has it). You learn about the tidyverse and how to use ggplot and how to make shiny apps (which introduces you to server–client frameworks) and how to make an R package (didn't even know you could do that!). James is great, and I would highly recommend
