The best AM class I have taken so far by a lot!
Lin alg background is very good to have in the course, if you haven't taken it yet you should review concepts before taking this class, it will help a lot with your intuition! This course is very applicable to various fields where optimization is used, especially in forming the foundation of what I imagine is much of industrial engineering. But in any situation, such as a match–making in a market, I can think of where an LP model could be useful to solve for optimal amounts of each good / optimal allocations as long as we have enough information about the relationships. This course changed the way I saw the world and provided me a mathematical framework to think about modeling problems in the world, from social to production problems. This is a canonical class for AM/Ec concentrators –– you feel like you start to understand the heart of applied economics after you've taken this. An accessible, reasonably paced Applied Math course with really interesting real–world applications. Not too difficult compared to some other options for AM breadth distribution requirements, but a fair amount of time required prior to each class given that the course adopts a flipped classroom style.
I think this class is a great introduction to foundational ideas in optimization. The class spends a large amount of time specifically on Linear Programming (i.e. how to maximize or minimize a linear objective function with linear constraints), and some time on Integer Programming towards the end. There is a very brief mention of Convex Optimization at the end but this course really doesn't go into detail for that area of optimization and it is sufficiently different from Linear/Integer programming that I'd say this course does not cover it. That said, it is a very good introduction to Linear and Integer programming. It explores the Simplex Algorithm in detail, which is the canonical algorithm for solving linear programming problems. It builds the algorithm step–by–step and gives lots of practice with applying the algorithm. The course also develops skills with AMPL, which is "A Mathematical Programming Language" used for optimization problems. This allows for practical skills to accompany the theoretical concepts. That said, the course is very much an introductory one and I'm not certain there are many other courses that build on the material in this course so it may not have much of a sequence to it.

The work load is quite manageable; a p–set every other week and quizzes during the weeks without p–sets. There wasn't a final and the lowest quiz score is dropped as long as you score above 50% in all quizzes. One caveat I'd be aware of is that this class is apparently well known to be not too heavy in workload so a lot of athletes take it and if you get paired with athletes for the mini–projects, you may find that they don't do any work which can be a really severe negative experience.
